## 🧠 About Me

I am a backend-focused developer interested in building practical AI systems, especially around **RAG**, **document intelligence**, **data ingestion**, and **LLM-powered applications**.

I like working close to the engine room: APIs, parsers, chunkers, retrieval pipelines, vector databases, databases, caching layers, and deployment workflows.

Currently, I am exploring how modern AI tools like **Claude Code**, **Codex**, and LLM-based development workflows can help build better software without losing engineering discipline.
